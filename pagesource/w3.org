         <h3 class="h4 tPadding0 bPadding0 summary">
            <span class="expand_section">WOFF File Format 2.0 is now a W3C Recommendation</span>
         </h3>
         <p class="date">
            <span class="dtstart published" title="2018-03-01T11:21:10Z"> 1 March 2018 </span> | <a title="Archive: WOFF File Format 2.0 is now a W3C Recommendation" href="https://www.w3.org/blog/news/archives/6846">Archive</a>
         </p>
      </div>
      <div class="description expand_description"><p><a class="imageLink" href="https://www.w3.org/2018/02/media-advisory-woff2-rec.html.en"><img src="https://www.w3.org/Fonts/Style/fontsbg.png" alt="fonts illustration" width="100" /></a>The <a href="https://www.w3.org/Fonts/WG/">WebFonts Working Group</a> has published <a href="https://www.w3.org/TR/2018/REC-WOFF2-20180301/">WOFF File Format 2.0</a> as a W3C Recommendation. Based on experience with WOFF 1.0, which is widely deployed, this specification was developed to provide improved compression and thus lower use of network bandwidth, while still allowing fast decompression even on mobile devices. This is achieved by combining a content-aware preprocessing step and improved entropy coding, compared to the Flate compression used in WOFF 1.0. Read more in the <a href="https://www.w3.org/2018/02/media-advisory-woff2-rec.html.en">W3C Media Advisory</a>.</p>
</div>

         <h3 class="h4 tPadding0 bPadding0 summary">
            <span class="expand_section">Call for Review: CSS Color Module Level 3 is a W3C Proposed Recommendation</span>
         </h3>
         <p class="date">
            <span class="dtstart published" title="2018-03-15T09:18:25Z"> 15 March 2018 </span> | <a title="Archive: Call for Review: CSS Color Module Level 3 is a W3C Proposed Recommendation" href="https://www.w3.org/blog/news/archives/6906">Archive</a>
         </p>
      </div>
      <div class="description expand_description"><p>The <a href="https://www.w3.org/Style/CSS/members">CSS Working Group</a> has published a Proposed Recommendation of <a href="https://www.w3.org/TR/2018/PR-css-color-3-20180315/">CSS Color Module Level 3</a>. CSS (Cascading Style Sheets) is a language for describing the rendering of HTML and XML documents on screen, on paper, in speech, etc. It uses color-related properties and values to color the text, backgrounds, borders, and other parts of elements in a document. This specification describes color values and properties for foreground color and group opacity. These include properties and values from CSS level 2 and new values.</p>
<p>Comments are welcome through <strong>12 April 2018</strong>.</p>
</div>

         <h3 class="h4 tPadding0 bPadding0 summary">
            <span class="expand_section">W3C Invites Implementations of the CSS Fonts Module Level 3</span>
         </h3>
         <p class="date">
            <span class="dtstart published" title="2018-03-15T09:16:47Z"> 15 March 2018 </span> | <a title="Archive: W3C Invites Implementations of the CSS Fonts Module Level 3" href="https://www.w3.org/blog/news/archives/6904">Archive</a>
         </p>
      </div>
      <div class="description expand_description"><p>The <a href="https://www.w3.org/Style/CSS/members">CSS Working Group</a> invites implementations of an updated Candidate Recommendation of <a href="https://www.w3.org/TR/2018/CR-css-fonts-3-20180315/">CSS Fonts Module Level 3</a>. This CSS3 module describes how font properties are specified and how font resources are loaded dynamically. The contents of this specification are a consolidation of content previously divided into <a href="https://www.w3.org/TR/2002/WD-css3-fonts-20020802/">CSS3 Fonts</a> and <a href="https://www.w3.org/TR/2002/WD-css3-fonts-20020802/">CSS3 Web Fonts</a> modules. The description of font load events was moved into the <a href="https://drafts.csswg.org/css-font-loading-3/">CSS Font Loading</a> module.</p>
</div>

            <span class="expand_section">W3C Invites Implementations of the Timed Text Markup Language 2 (TTML2)</span>
         </h3>
         <p class="date">
            <span class="dtstart published" title="2018-03-13T07:49:09Z"> 13 March 2018 </span> | <a title="Archive: W3C Invites Implementations of the Timed Text Markup Language 2 (TTML2)" href="https://www.w3.org/blog/news/archives/6900">Archive</a>
         </p>
      </div>
      <div class="description expand_description"><p>The <a href="https://www.w3.org/AudioVideo/TT/">Timed Text Working Group</a> invites implementations of the <a href="https://www.w3.org/TR/2018/CR-ttml2-20180313/">Timed Text Markup Language 2 (TTML2)</a> Candidate Recommendation. This document specifies the Timed Text Markup Language (TTML), Version 2, also known as TTML2, in terms of a vocabulary and semantics thereof.</p>
<p>The Timed Text Markup Language is a content type that represents timed text media for the purpose of interchange among authoring systems. Timed text is textual information that is intrinsically or extrinsically associated with timing information.</p>

         <h3 class="h4 tPadding0 bPadding0 summary">
            <span class="expand_section">ACT Rules Format 1.0 Working Draft Updated</span>
         </h3>
         <p class="date">
            <span class="dtstart published" title="2018-03-08T12:37:58Z"> 8 March 2018 </span> | <a title="Archive: ACT Rules Format 1.0 Working Draft Updated" href="https://www.w3.org/blog/news/archives/6889">Archive</a>
         </p>
      </div>
      <div class="description expand_description"><p>An updated Working Draft of <a href="https://www.w3.org/TR/2018/WD-act-rules-format-1.0-20180308/">Accessibility Conformance Testing (ACT) Rules Format</a> was published today by the <a href="https://www.w3.org/WAI/GL/">Accessibility Guidelines Working Group (AG WG)</a>. This draft addresses all comments received on the previous draft. Most importantly, the &#x201C;Selector&#x201D; approach was replaced with &#x201C;Applicability&#x201D; and &#x201C;Expectation&#x201D; pairs, to make ACT Rules less procedural. This draft is accompanied by <a href="https://w3c.github.io/wcag-act-rules/">sample ACT Rules</a> that implement this rules format. Read about the the <a href="https://www.w3.org/WAI/">Web Accessibility Initiative (WAI)</a>.</p>

            <span class="expand_section">W3C Opens Technical Architecture Group (TAG) Special Election</span>
         </h3>
         <p class="date">
            <span class="dtstart published" title="2018-03-05T11:19:56Z"> 5 March 2018 </span> | <a title="Archive: W3C Opens Technical Architecture Group (TAG) Special Election" href="https://www.w3.org/blog/news/archives/6871">Archive</a>
         </p>
      </div>
      <div class="description expand_description"><p><a class="imageLink" href="https://www.w3.org/2018/03/05-tag-nominations"><img src="https://www.w3.org/blog/news/files/2017/12/w3c-tag-300x300.jpg" alt="W3C TAG logo" width="80" height="80" /></a>The W3C Advisory Committee having <a href="https://www.w3.org/2018/03/05-tag-nominations">nominated</a> three individuals, is invited today to vote until 2 April 2018 for one seat&#xA0;in the 2018 special election to the W3C Technical Architecture Group (TAG) per <a href="https://www.w3.org/2018/Process-20180201/#changes">Process Document now granting an additional seat</a>.</p>
<p>The TAG is a special working group within the W3C, chartered under the <a href="https://www.w3.org/2018/Process-20180201/#TAG">W3C Process Document</a>, with stewardship of the Web architecture. Some aspects of its mission include</p>

to interpret and clarify these principles when necessary;</li>
<li>to resolve issues involving general Web architecture brought to the
TAG;</li>
<li>to help coordinate cross-technology architecture developments inside and
outside W3C.</li>

         <h3 class="h4 tPadding0 bPadding0 summary">
            <span class="expand_section">Call for Review: TTML Profiles for Internet Media Subtitles and Captions 1.0.1 (IMSC1) is a W3C Proposed Recommendation</span>
         </h3>
         <p class="date">
            <span class="dtstart published" title="2018-02-27T17:06:42Z"> 27 February 2018 </span> | <a title="Archive: Call for Review: TTML Profiles for Internet Media Subtitles and Captions 1.0.1 (IMSC1) is a W3C Proposed Recommendation" href="https://www.w3.org/blog/news/archives/6850">Archive</a>
         </p>
      </div>
      <div class="description expand_description"><p>The <a href="https://www.w3.org/AudioVideo/TT/">Timed Text Working Group</a> has published a Proposed Recommendation of <a href="https://www.w3.org/TR/2018/PR-ttml-imsc1.0.1-20180227/">TTML Profiles for Internet Media Subtitles and Captions 1.0.1 (IMSC1)</a>. This document specifies two profiles of [<a href="https://www.w3.org/TR/2018/PR-ttml-imsc1.0.1-20180227/#bib-TTML1">TTML1</a>]: a text-only profile and an image-only profile. These profiles are intended to be used across subtitle and caption delivery applications worldwide, thereby simplifying interoperability, consistent rendering and conversion to other subtitling and captioning formats.</p>
<p>Comments are welcome through <strong>27 March 2018</strong>.</p>
